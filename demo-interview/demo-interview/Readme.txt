This project has an api that shows the metadata of movie directors and querying the same by id it
shows data of respective director movies.

Technologies used :

Spring boot framework
H2 database

Task :
 1. Bring the service up in your local machine by fixing the issues.
 2. Ensure that the two api's that are exposed are working fine.
 3. Create an api which takes in the director name and gives out the movie details.
 4. Create an api which takes in the director name and gives out the image or link to an image of the selected director.
